
# Project 1: Number Prediction (Guess the Number) 🎯
**Author: Suraj Poddar**

A simple and interactive Python-based game where the computer selects a random number, and the player tries to guess it. The program provides real-time feedback and handles input errors gracefully.



## 📋 Project Goal
The goal of this project is to demonstrate fundamental programming concepts, including:
* **Randomization:** Using the `random` library to generate unpredictable outcomes.
* **Control Flow:** Utilizing `while` loops and `if-elif-else` statements.
* **Error Handling:** Using `try-except` blocks to prevent the program from crashing on invalid inputs.
* **Functional Programming:** Wrapping the game logic inside a `main()` function.

## 🚀 How to Run
1. Ensure you have **Python 3.x** installed.
2. Download or clone the file `number_prediction.py`.
3. Run the script using your terminal:
   ```bash
   python number_prediction.py

```

## 🕹️ How to Play

1. The computer picks a secret number between **1 and 100**.
2. Type your guess into the terminal.
3. If your guess is too high or too low, the program will hint at the direction.
4. The game ends when you guess the correct number!

## 📝 Code Logic Explained

* **`import random`**: Imports the library needed for random number generation.
* **`while guess != number`**: Keeps the game running until the correct answer is found.
* **`try / except`**: Ensures that if a user types text instead of a number, the game doesn't crash.
* **`break`**: Stops the loop immediately once the "Congratulations" message is printed.
